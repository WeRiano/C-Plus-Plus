1/**
2* @file
3* @brief Implementation of the [Hyperplane Separation Theorem]
4* (https://en.wikipedia.org/wiki/Hyperplane_separation_theorem) (or
5* Separating Axis Theorem) for detecting collisions in 2D-space.
6* @details
7* Implements [Hyperplane Separation Theorem]
8* (https://en.wikipedia.org/wiki/Hyperplane_separation_theorem) (or
9* Separating Axis Theorem) to detect convex polygon-polygon and convex
10* polygon-circle collision.
11*
12* Two convex objects do not overlap if there exists a line (called axis) onto
13* which the two objects' projections do not overlap. The axis is a separating
14* axis if an orthogonal straight line (called the separating line) can be drawn
15* between the objects without colliding with either.
16*
17* The algorithm should run in O(n) time for polygon-X collisions and O(1) for
18* circle-circle collisions, where n is the largest number of vertices that
19* belongs to one of the polygons
20
21* @author [William Fors](https://github.com/WeRiano)
22*/
